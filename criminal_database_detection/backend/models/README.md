# AI Models Directory

Place the following pretrained model files in this directory:

1. `deploy.prototxt` — OpenCV SSD face detector architecture
2. `res10_300x300_ssd_iter_140000.caffemodel` — SSD pretrained weights
3. `arcface_r100.onnx` — ArcFace R100 face recognition model

See setup_guide.md for download instructions.
